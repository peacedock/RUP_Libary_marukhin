# Глоссарий проекта
| Термин | Что означает |
|--------|--------------|
| Читатель | Студент или сотрудник техникума, который берет книги |
| Книга | Издание, которое есть в библиотеке (может быть в неск. экземплярах) |
| Экземпляр | Конкретный физический книжный экземпляр | 
| Бронь | Временное закрепление книги за читателем (обычно на 2 дня) | 
| Выдача | Передача книги читателю на срок |
| Задолженность | Просрочка возврата книги | 
| Каталог | Список всех книг с возможностью поиска |
